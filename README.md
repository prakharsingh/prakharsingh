### Prakhar Singh · Founding Engineer @ DubaiDeals.live · ex-VP Engineering @ Deutsche Bank (DWS)

Principal engineer and engineering leader with 12+ years across FinTech,
EdTech, and E-commerce. I split my time between building hands-on and leading
teams; my recent focus is applied AI: agentic systems, retrieval-grounded
LLMs, and edge architectures.

At **Deutsche Bank (DWS)** I built and led a 30-person engineering team across
India, the UK, and Spain behind an AI/ML trading-automation platform
**credited with ~€35M annual revenue uplift**. We rolled out LLM copilots
(~25% developer-throughput gain) and a Sentry/Datadog observability stack
(~40% MTTR reduction).

My current work is **DubaiDeals.live**, an edge-first consumer platform that
serves from Cloudflare Workers with no database at request time; its "Ask
Savyt" assistant is retrieval-first, so the model parses intent only while
deterministic code ranks and renders results. On a 2026 contract engagement
with **Devzy AI** I architected an agentic code-review system: multi-model
LLM orchestration combined with static analysis, test execution, and
compliance-aware security checks (GDPR, HIPAA, PCI-DSS, SOC 2 aligned),
shipped across CLI, PR, and IDE integrations.

Based in Pune, Maharashtra, India, relocating to Sweden. Works remotely with
global teams.
